Announcing Presentations, Speakers, and Schedule¶

Our talk selection process has just wrapped up and we’re eager to announce the presentations for this year’s conference. Each year we select talks that emphasize intersections between tech writers, developers, designers, and translators, because we all have so much to learn from each other.

Schedule¶

The high-level schedule for the conference is now available here. We will post details about venues and final talk assignments in the next few weeks.

Presentations¶

Here is the full list of presentations for Write the Docs Europe 2015:

  • Adam Dangoor - Tested and Correct, How to Make Sure Your Documentation Keeps Working
  • Anna Jaruga - The quest for scientific credit for software documentation
  • Beth Aitman - A crash course in writing for user interfaces
  • Chris Ward - Documenting your Story - Crafting a good presentation
  • Christina Elmore - All roads might not lead to docs
  • Diana Potter - Screencasting 101
  • Elijah Caine - How to Write an Email
  • Emilie Boillat - Whatchamacallit: Controlled Vocabularies for Technical Writers
  • Florian Scholz & Jean-Yves Perrier - Gardening Open Docs
  • Jamie Hannaford - Generating docs from APIs
  • Jennifer Rondeau - Back to the Future: What Can Documentarians Learn From The Past?
  • Markus Zapke-Gründemann - i18what?
  • Patrick Keegan - Free Your Mind and Your Docs Will Follow
  • Paul Adams - Judas Priest Ate My Scrum Master
  • Paul Roeland - MacGyvering your docs
  • Radina Matic - Inclusive Tech Docs - TechComm Meets Accessibility
  • Riona MacNamara - IMPOSTER NO MORE: How Tech Writers Can Shed Self-Doubt, Embrace Uncertainty, and Surf the Upcoming Swerve in Technical Documentation
  • Sonja Heinen - Visual Documentation Language

You can check out the full talk abstracts on our website.
